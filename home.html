Meus Principais conhecimentos

Algumas maravilhosas tecnologias que aprendi nesse meu tempo na programação!

HTML5 tenho um pleno conhecimento pois é uma das linguagens que uso em práticamente tudo que faço, procuro sempre ser o mais sintático possível.

CSS3 essa linguagem tão odiada e tão adorada ao mesmo tempo, eu costumo fazer um parelelo ao português por que estou constantemente aprendendo coisas novas e evoluindo e quanto mais aprendo mais vou tomando gosto.

Javascript/ES6 essa aqui é bem maneira gostei muito, pois comecei com o C como primeira linguagem pular para o JS foi uma maravilha por sua facilidade e modernidade, costumo usar as últimas features da linguagem e focar no clean code.

Node.js aprendi esta linguagem por que quando comecei pelo frontend não estava gostando muito de ter apenas dados estáticos quis aprender como colocar dados dinamicamente e Node foi o cara da vez.

ReactJS junto ao Node foi o escolhido para criar paginas com mais facilidade uma tecnologia tão fantástica que é muito mão na roda, que ajuda demais no desenvolvimento.

React Native Não é uma das tecnologias com que brinco tanto atualmente, mas já tive um contanto legal e gostei bastante.

Mongo DB utilizo bastante em meus projetos por sua facilidade e desempenho apesar de não ter um conhecimento muito grande, ou seja estou sempre nas Docs consultando algo.

MySQL essa é uma linguagem que tenho um conhecimento legal dos comandos para fazer um CRUD mais não é uma linguagem que costumo integrar tanto em meus projetos Node.

Bootstrap se tem algo mão não roda mesmo é esse cara, gosto bastante de usar ajuda muito na responsividade, porém não gosto de ser refém dele costumo brincar com o CSS3 para mudar um pouco.
Obs.: esse site é feito com CSS3 puro ;)

Principais projetos solos

Meus principais projetos que realizei nessa minha caminhada na programação, adorei fazer cada um deles consegui adquirir muito conhecimento e me diverti bastante no processo.

Book On Table

Um dos meus projetos mais queridos, pois foi um dos mais difíceis e complexos que fiz e um dos que mais me animou, a ideia desse projeto foi basicamente prover conteúdo para quem deseja aprender Inglês, e o foco do conteúdo são pequenos trechos de séries, o conteúdo é constituído de um trecho do vídeo de determinada série, logo abaixo possuí o texto do vídeo em Inglês e a tradução ao lado.

Este projeto possuí duas partes, uma que é a parte admin com autenticação onde é possível administrar o conteúdo e outra parte onde qualquer um pode entrar o consumir o conteúdo disponibilizado.

Tecnologias: Node.js, handlebars, Bootstrap, MongoDB.

Features: Autenticação, Uploads, API Rest, Paginação.

Cópia do OLX

Projeto feito com a intenção de fazer algo similar ao OLX onde o usuário se cadastra e então pode realizar login e logout, e também cadastrar produtos para exibir, e os gerenciar (editar, excluir).

O projeto se constituí de uma API Rest feita com Node.js e o Frontend feito em ReactJS consumindo essa API junto ao axios.

Tecnologias: Node.js, ReactJS, Bootstrap, MongoDB,.

Features: Uploads, API Rest, Paginação.

App de To DO

Um dos meus primeiros projetos "full stack" este aqui fiz logo quando aprendi a usar ReactJs e conectar a aplicação ao Banco de Dados uma aplicação em si bem simples mais que para mim, foi algo fantástico quando completei.

O app funciona bem simples você cadastra seus afazeres diários pode marcar como completado, pode excluir, criar quantos quiser. O background que possuí no header atualiza de acordo com a hora, um app bem bacaninha.

Tecnologias: Node.js, ReactJS, React Native, MySql.

Features: CRUD.
